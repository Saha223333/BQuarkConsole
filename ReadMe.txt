Программа BConsole. Является конструктором файлов заданий для программы BQuark.
Позвоялет: опрашивать PLC счетчики Меркурий-230 по GSM-связи, хранить данные, делать отчеты. 

Комплекс состоит из частей: 

1.Непосредственно программы BConsole.exe
2.Программы опроса BQuark.3.0.exe
3.Базы данных на Oracle

Для корректной работы комплекса требуется:

1.Убедиться что файлы BQuark.3.0.exe и BQuark.ini находятся по пути "C:\BQuark\BQuark.3.0.exe" и "C:\BQuark\BQuark.ini"
2.Убедиться что в файле BQuark.ini есть строка [AutoExportCSV] и строка TIMESTAMP=NO после нее.
3.Если у пользователя нет прав администратора, установить у BQuark.3.0.exe режим совместимости с Windows 2000
4.Модем 

Горячие клавиши:

F1 - Форма отчетов
F2 - Форма заданий
Ctrl-Z - Удалить все, кроме выбранного узла
Ctrl-X - Очистить расписание
Ctrl-D - Галочка добавления DEXT при генерации задания
Ctrl-F - Открыть форму поиска
